# Hata Yönetimi

## Kapsam
- Exception (istisna) kavramı
- try, except, finally blokları
- Farklı hata türleri (ValueError, TypeError, vs.)
- Özel exception oluşturma
- Hata mesajları ve debugging
- assert kullanımı
- Logging (kayıt tutma)
- Defensive programming

## Öğrenme Hedefleri
- Programları hatalara karşı dayanıklı hale getirmek
- Hata ayıklama becerisi geliştirmek
- Kullanıcı dostu hata mesajları oluşturmak

## Örnek Dosyalar
- `temel_hata_yonetimi.py` - try/except kullanımı
- `hata_turleri.py` - Farklı exception türleri
- `ozel_hatalar.py` - Custom exceptions
- `debugging_teknikleri.py` - Debug yöntemleri
- `logging_kullanimi.py` - Log kayıtları
